SSL and TLS: Secure Protocols for Internet Communication


What is SSL?

SSL (Secure Sockets Layer) is an original protocol for secure internet communication. It was introduced by a company called Netscape in the mid-1990s. However, we no longer use this as a newer version is available, which we will discuss later.

How does it work?

SSL does its job by combining public key and symmetric key encryption, which we have previously discussed, between one client to a server. It allows the two to verify each other's identities and establish a safe connection for transmitting data, including private information that you don’t want to expose to a hacker or an imposter.

What is TLS?

This right here is the reason why we don’t use SSL anymore. TLS (Transport Layer Security), which we often call it as a successor when compared to SSL, is SSL's bigger brother. TLS does the same job as SSL, but adding features makes it more reliable. Since its introduction at the end of the 1990s, it has evolved into the industry standard for safe online communication. However, we just called them both “SSL” because we don’t want to make it complicated.

How does it work?

TLS works just like SSL but with a very important feature called Forward secrecy. With this new feature, TLS can prevent decryption of previous communication even if a server's private key is compromised. It generates a new key for each TLS session, which is not derived from the server's private key. Below is a video showing how it works in detail! Make sure to check it out.

Benefits of SSL and TLS

There are many benefits when it comes to using SSL. Below are a few examples that I think are very notable.

  • Secure communication: SSL makes it possible to send sensitive data securely and safely, like passwords and credit card numbers.by encryption.
  • Authentication: SSL lets the client and server validate each other's identity, guaranteeing that the client connects with the intended server and not an impostor.
  • Standardization: TLS has replaced SSL as the primary protocol for secure internet communication and has become the standard for secure web communication.
  • Compatibility: SSL is accessible and widely used because they are compatible with the most of web browsers and operating systems.
